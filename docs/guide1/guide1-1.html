# Instalación

# Windows

Iremos a la pagina princial Visual Studio code (opens new window) y descargamos la version de windows que queramos version usuario o sistema.

# Linux

Buscamos en paqueteria si tenemos el programa.

apt-get search code

Si esta lo instalamos:

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install code
